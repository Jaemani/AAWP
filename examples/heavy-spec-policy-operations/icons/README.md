# Lucide badge icons

이 디렉터리는 상태 배지에 사용하는 Lucide 아이콘 `check`, `clock`,
`triangle-alert`, `shield-check`, `info`를 담는다. 원본 프로젝트와 라이선스는
[Lucide](https://lucide.dev/) 및 [ISC License](https://github.com/lucide-icons/lucide/blob/main/LICENSE)를 따른다.

아이콘은 외부 네트워크에 의존하지 않도록 demo snapshot에 포함하며 CSS mask로
semantic foreground token을 상속한다.
